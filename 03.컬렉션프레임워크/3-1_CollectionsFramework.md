# 컬렉션 프레임워크 (Collections Framework)

### 컬렉션?
: 여러 객체 (데이터) 를 모아둔 것

### 프레임워크란?
: 소프트웨어 개발을 수월하게 하기 위한 소프트웨어 개발 환경

### 컬렉션 프레임워크란?
: 컬렉션 (다수의 객체) 를 다루기 위한 표준화된 프로그래밍 방식

### 컬렉션 프레임워크의 특징
- 컬렉션을 쉽고 편하게 다룰 수 있는 다양한 클래스를 제공
- java.util 패키지에 포함되어있음
- JDK 1.2 부터 제공됨

### 컬렉션 클래스란?
: 다수의 데이터를 저장할 수 있는 클래스
ex - Vector, ArrayList, HashSet
